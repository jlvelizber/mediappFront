# MediApp 

## Frontend - Made with NextJs 

## Product Scope

This frontend is a doctor-only application for independent private practices.

### In Scope

- Doctor authentication and session.
- Doctor dashboard and daily agenda.
- Patient management (doctor-owned patients).
- Appointments, medical records, and prescriptions.
- Doctor availability and professional profile.

### Out of Scope (handled in other products)

- Patient portal and patient self-service flows.
- Admin backoffice for multi-doctor organizations.
- Clinic/medical center management.
- Billing, subscriptions, and customer management.

### Architecture Rule

Keep this project focused on a single role and context: one independent doctor using their private office workflow.
